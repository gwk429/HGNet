# HGNet: A High-Order Geometric Consistency Network for Robust Point Cloud Registration

PyTorch implementation of the paper 'HGNet'

Weikang Gu, Li Xue, Heng Dong, Jiangtao Gao, Changcai Yang, Riqing Chen, Lifang Wei


1. 训练室内场景3DMatch数据集以及3DLoMatch数据集
run：python train_3DMatch.py
2. 训练室外场景KITTI数据集
run：python train_KITTI.py
3. 测试3DMatch,3DLoMatch,KITTI分别
run: python test_3DMatch.py, test_3DLoMatch.py, test_KITTI.py
